# Portfolio

Personal developer portfolio for Dylan — final-year IT student (Belgium Campus ITversity), IT intern, and freelance web developer based in Centurion, South Africa.

Built with plain HTML, CSS, and vanilla JavaScript. No frameworks, no build tools. Hosted free on GitHub Pages.

## Structure

```
.
├── index.html      # Page markup
├── css/            # Styles
├── js/             # Behaviour
└── assets/         # Icons / static files
```

## Run locally

Open `index.html` in any browser, or serve the folder with a static server.

## Deploy

Hosted via GitHub Pages from the `main` branch.
